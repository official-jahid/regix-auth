🚀 Getting Started

REGIX Auth System provides HWID-locked authentication for your applications. Choose from three authentication methods:

  • License Keys — Simple key-based auth
  • User Accounts — Username + Password + HWID
  • Plan Licenses — Subscription-based with plans (basic, premium, enterprise)

Base URL: http://your-server:3000

All authentication endpoints accept JSON POST requests.

🔑 Key Authentication

For DLL, C++, C#, PowerShell, and Web apps using license keys.

Login with Key + HWID

POST /api/auth/login 

{
  "key": "REGIX-A1B2C3D4E5F6",
  "hwid": "YOUR-MACHINE-HWID"
}

Response (Success):

{
  "valid": true,
  "message": "Device activated successfully!",
  "expiry": 1735689600
}

Verify Key + HWID

POST /api/auth/verify 

{
  "key": "REGIX-A1B2C3D4E5F6",
  "hwid": "YOUR-MACHINE-HWID"
}

Check Key Status (no HWID)

POST /api/auth/check 

{
  "key": "REGIX-A1B2C3D4E5F6"
}

Register New Device

POST /api/auth/register-device 

{
  "key": "REGIX-A1B2C3D4E5F6",
  "hwid": "NEW-DEVICE-HWID"
}

Remove Device

POST /api/auth/remove-device 

{
  "key": "REGIX-A1B2C3D4E5F6",
  "hwid": "DEVICE-HWID-TO-REMOVE"
}

👤 User Authentication

For applications using username/password accounts with HWID locking.

Login with Username + Password + HWID

POST /api/auth/login 

{
  "user": "myusername",
  "pass": "mypassword",
  "hwid": "YOUR-MACHINE-HWID"
}

Check Expiry

POST /api/auth/expiry 

{
  "user": "myusername",
  "pass": "mypassword"
}

Response: {"found": true, "expiry": 1735689600, "status": "active"}

📜 License (Plan) Authentication

For subscription-based licensing with plans.

Activate License

POST /api/license/activate 

{
  "license_key": "REGIX-A1B2C3D4E5F6",
  "hwid": "YOUR-MACHINE-HWID",
  "product": "REGIX-Auth"
}

Validate License

POST /api/license/validate 

{
  "license_key": "REGIX-A1B2C3D4E5F6",
  "hwid": "YOUR-MACHINE-HWID"
}

🔄 HWID Reset

Allow users to reset their HWID to activate on a new device.

Request HWID Reset

POST /api/hwid/reset-request 

{
  "username": "myusername",
  "hwid": "OLD-HWID-TO-RESET",
  "reason": "New computer"
}

Response: {"success": true, "message": "HWID has been reset. You can now log in from your new device."}

🔐 Forgot Password

Reset user password via the secret key (admin/bot only).

Reset Password

POST /api/auth/forgot-password?secret=YOUR_SECRET_KEY 

{
  "username": "myusername",
  "newPassword": "newpassword123"
}

⚙️ Admin Management API

All admin endpoints require the secret key as query parameter or in body.

Create User

POST /api/admin/create-user?secret=YOUR_SECRET_KEY 

{
  "username": "newuser",
  "password": "userpass",
  "email": "user@example.com",
  "duration": 720,
  "max_devices": 1
}

Create Key

POST /api/admin/create-key?secret=YOUR_SECRET_KEY 

{
  "duration": 720,
  "max_devices": 1,
  "note": "Premium key"
}

Create License

POST /api/admin/create-license?secret=YOUR_SECRET_KEY 

{
  "plan": "premium",
  "product": "REGIX-Auth",
  "duration_days": 365,
  "max_devices": 3
}

Update Status (Ban/Freeze/Activate)

POST /api/admin/update-status?secret=YOUR_SECRET_KEY 

{
  "type": "user",
  "id": 1,
  "status": "banned"
}

Types: key, user, license | Statuses: active, banned, frozen, expired

Delete Entry

POST /api/admin/delete?secret=YOUR_SECRET_KEY 

{
  "type": "user",
  "id": 1
}

Blacklist HWID

POST /api/admin/blacklist?secret=YOUR_SECRET_KEY 

{
  "hwid": "BAD-HWID-HERE",
  "reason": "Abuse detected"
}

List All Items

GET /api/bot?secret=YOUR_SECRET_KEY&action=list&type=keys

Types: keys, users, licenses, blacklist, apis, logs

👥 Panel User API

For the web dashboard login and user management.

Panel Login

  "password": "RegixAdmin123!"
}

Get Current User

GET /api/panel/me (Requires session/Bearer token)

Create Panel User (Owner only)

POST /api/panel/users 

{
  "username": "mod1",
  "password": "securepass",
  "email": "mod@regix.com",
  "role": "moderator"
}

Roles: owner, developer, moderator, reseller, supporter, user
